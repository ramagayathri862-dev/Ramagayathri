Portfolio Home Page Description

  1. This page is created as the home page of my personal portfolio website.
  2. HTML is used to structure the webpage elements.
  3. CSS is used for styling and designing the layout of the page.
  4. A background image is added using CSS to improve the visual appearance.
  5. A navigation bar is created with links such as Home, Skills, Projects, and Contact.
  6. Flexbox is used in CSS to align the navigation menu in the center.
  7. A container section is used to organize the main content of the webpage.
  8. A circular profile image is created using the border-radius property.
  9. The object-fit property is used to properly fit the profile image inside the circle.
  10. A box shadow is applied to the image to give a better visual effect.
  11. Headings and paragraphs are used to introduce the portfolio and personal information.
  12. CSS styling such as font-family, margin, and text alignment is used for better readability.
  13. A hover effect is applied to navigation links to improve user interaction.
  14. A footer section is added at the bottom of the page.
  15. The footer is fixed at the bottom using CSS positioning.
